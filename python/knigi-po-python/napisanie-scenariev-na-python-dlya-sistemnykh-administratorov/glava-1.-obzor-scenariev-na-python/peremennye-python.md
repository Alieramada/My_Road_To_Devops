# Переменные Python

В Python _**нет необходимости**_ сначала объявлять переменные. В Python просто придумайте любое имя для своей переменной и присвойте ей значение. Вы можете использовать эту переменную в своей программе. Таким образом, в Python вы можете объявлять переменные, когда они вам понадобятся.

В Python значение переменной может меняться во время выполнения программы, так же как и ее тип. В следующей строке кода мы присваиваем переменной значение 100:

```
n = 100
Here are assigning 100 to the variable n. Now, we are going to increase the value of n by 1:
>>> n = n + 1
>>> print(n)
101
>>>
```

Ниже приведен пример типа переменной, который может изменяться во время выполнения:

```
a = 50 # data type is implicitly set to integer
a = 50 + 9.50 # data type is changed to float
a = "Seventy" # and now it will be a string
```

Python заботится о представлении различных типов данных, то есть значения каждого типа хранятся в разных ячейках памяти. Переменной будет имя, которому мы собираемся присвоить значение:

```
>>> msg = 'And now for something completely different'
>>> a = 20
>>> pi = 3.1415926535897932
```

В этом примере выполняется три присваивания. Первое присваивание - это присваивание строки переменной с именем msg. Второе присваивание - это присваивание целого числа переменной с именем a, а последнее присваивание - это присваивание значения pi.

Тип переменной - это тип значения, на которое она ссылается. Посмотрите на следующий код:

```
>>> type(msg)
<type 'str'>
>>> type(a)
<type 'int'>
>>> type(pi)
<type 'float'>
```

## &#x20;Создание и присвоение значений переменным

В Python переменные не нужно объявлять явно, чтобы зарезервировать место в памяти. Таким образом, объявление выполняется автоматически всякий раз, когда вы присваиваете значение переменной. В Python знак равенства = используется для присвоения значений переменным.

Рассмотрим следующий пример:\


```
#!/usr/bin/python3
name = 'John'
age = 25
address = 'USA'
percentage = 85.5
print(name)
print(age)
print(address)
print(percentage)

Output:
John
25
USA
85.5
```

В предыдущем примере переменной "name" мы присвоили значение "John", переменной "age" - значение "25", переменной "address" - значение "USA", а переменной "percentage" - значение "85,5".

Нам не нужно сначала объявлять их, как это делается в других языках. Итак, при просмотре значения интерпретатор получит тип этой переменной. В предыдущем примере имя и адрес были строками, возраст - целым числом, а процент - числом с плавающей запятой.

Множественное присваивание одного и того же значения может быть выполнено следующим образом:

```
x = y = z = 1
```

В предыдущем примере мы создали три переменные и присвоили им целочисленное значение 1, и все эти три переменные будут отнесены к одной и той же ячейке памяти.

В Python мы можем присвоить несколько значений нескольким переменным в одной строке:

```
x, y, z = 10, 'John', 80
```

Здесь мы объявили одну строковую переменную y и присвоили ей значение John, а также две целочисленные переменные x и z, присвоив им значения 10 и 80 соответственно.\

## &#x20;Числа

Интерпретатор Python также может работать как калькулятор. Вам просто нужно ввести выражение, и оно вернет значение. Круглые скобки ( ) используются для группировки, как показано в следующем примере:

```
>>> 5 + 5
10
>>> 100 - 5*5
75
>>> (100 - 5*5) / 15
5.0
>>> 8 / 5
1.6
```

Целые числа относятся к типу int, а дробные - к типу float

{% hint style="info" %}
В Python операция деления (/) всегда возвращает значение с плавающей запятой. При делении на уровни (//) получается целочисленный результат. Оператор % используется для вычисления остатка.&#x20;
{% endhint %}

Рассмотрим следующий пример:

```
>>> 14/3
4.666666666666667
>>>
>>> 14//3
4
>>>
>>> 14%3
2
>>> 4*3+2
14
>>>
>>> 14%3
2
>>> 4*3+2
14
>>>
```

Для вычисления степеней в Python есть оператор \*\*, как показано в следующем примере:

```
>>> 8**3
512
>>> 5**7
78125
>>>
```

Знак равенства (=) используется для присвоения значения переменной:

```
>>> m = 50
>>> n = 8 * 8
>>> m * n
3200
```

Если переменная не имеет никакого значения и мы все равно попытаемся ее использовать, то интерпретатор выдаст сообщение об ошибке:

```
>>> k
Traceback (most recent call last):
File "<stdin>", line 1, in <module>
NameError: name 'k' is not defined
>>>
```

Если операторы имеют смешанные типы операндов, то значение, которое мы получим, будет иметь плавающую точку:

```
>>> 5 * 4.75 - 1
22.75
```

В интерактивной консоли Python значение \_ содержит последнее напечатанное значение выражения, как показано в следующем примере:

```
>>> a = 18.5/100
>>> b = 150.50
>>> a * b
27.8425
>>> b + _
178.3425
>>> round(_, 2)
178.34
>>>
```

Числовые типы данных хранят числовые значения, которые являются неизменяемыми типами данных. Если мы сделаем это, Python выделит новый объект для измененного типа данных.

Мы можем создавать числовые объекты, просто присваивая им значение, как показано в следующем примере:

```
num1 = 50
num2 = 25
```

Оператор del используется для удаления одной или нескольких переменных. Обратитесь к следующему примеру:

```
del num
del num_a, num_b
```

## &#x20;Преобразование числовых типов

В некоторых ситуациях вам необходимо явно преобразовать число из одного типа в другой, чтобы удовлетворить некоторым требованиям. Python выполняет это внутри выражения.

Введите int(a), чтобы преобразовать a в целое число&#x20;

Введите float(a), чтобы преобразовать a в число с плавающей запятой&#x20;

Введите complex(a), чтобы преобразовать a в комплексное число с действительной частью x и мнимой частью, равной нулю&#x20;

Введите complex(a, b), чтобы преобразовать a и b в комплексное число с действительной частью a и мнимой частью b. a и b являются числовыми выражениями

## Строки

Подобно числам, строки также являются одной из структур данных в Python. Python может манипулировать строками. Строки могут быть выражены следующим образом:

Заключены в одинарные кавычки ('...')&#x20;

Заключены в двойные кавычки ("...")

Смотрите следующий пример:

```
>>> 'Hello Python'
'Hello Python'
>>> "Hello Python"
'Hello Python'
```

Строка - это набор символов. Мы можем обращаться к символам по одному, следующим образом:

```
>>> city = 'delhi'
>>> letter = city[1]
>>> letter = city[-3]
```

Во втором предложении мы выбираем символ с номером 1 из city и присваиваем ему значение letter. Число в этих квадратных скобках является индексом. Индекс указывает, к какому символу вы хотите получить доступ. Они начинаются с 0. Итак, в предыдущем примере, когда вы выполните команду letter = city\[1], вы получите следующий результат:

```
city d e l h i
index 0 1 2 3 4
-5 -4 -3 -2 -1

Output:
e
l
```

## Конкатинация(+) и дублирование(\*) строк

Обратитесь к следующему коду:

```
>>> 3 * 'hi' + 'hello'
'hihihihello'
```

В предыдущем примере мы выполняем конкатинацию строк и их повторение. 3 \* "hi" означает, что "hi" печатается 3 раза, и, используя знак +, мы присоединяем строку "hello" рядом с ним.

Мы можем автоматически объединить две строки, просто написав их рядом друг с другом. Эти две строки должны быть заключены в кавычки, как показано здесь:

```
>>> 'he' 'llo'
'hello'
```

Эта функция действительно полезна, когда у вас длинные строки и вы хотите их разорвать. Вот пример:

```
>>> str = ('Several strings'
... 'joining them together.')
>>> str
'Several strings joining them together.'
```

## &#x20;Слайсинг строк

Строки поддерживают разбиение на части, что означает получение символов в заданном диапазоне из вашей строки. Давайте рассмотрим следующий пример. Обратите внимание, что начальное значение индекса всегда включается, а конечное значение всегда исключается.

Рассмотрим строку, str = "Programming":

```
>>> str[0:2]
'Pr'
>>> str[2:5]
'ogr
```

Теперь значение по умолчанию для пропущенного первого индекса равно нулю, как в примере:

```
>>> str[:2] + str[2:]
'Python'
>>> str[:4] + str[4:]
'Python'
>>> str[:2]
'Py'
>>> str[4:]
'on'
>>> str[-2:]
'on'
```

## &#x20;Доступ к значениям в строках

Мы можем получить доступ к символам из строк, используя разбиение на части с помощью квадратных скобок. Мы также можем получить доступ к символам из строк в пределах указанного диапазона. Обратитесь к следующему примеру:

```
#!/usr/bin/python3
str1 = 'Hello Python!'
str2 = "Object Oriented Programming"
print ("str1[0]: ", str1[0])
print ("str2[1:5]: ", str2[1:5])

Output:
str1[0]: H
str2[1:5]: bjec
```

## &#x20;Обновление строк

Мы можем обновить строку, присвоив указанному индексу новое значение. Обратитесь к следующему примеру:

```
#!/usr/bin/python3
str1 = 'Hello Python!'
print ("Updated String: - ", str1 [:6] + 'John')

Output:
Updated String: - Hello John
```

## &#x20;Строки являются неизменяемыми

Строки являются неизменяемыми, что означает, что мы не можем изменять значения. Обратитесь к приведенному примеру:

```
>>> welcome = 'Hello, John!'
>>> welcome[0] = 'Y'
TypeError: 'str' object does not support item assignment
```

Поскольку строки неизменяемы, мы не можем изменить существующую строку. Но мы можем создать новую строку, которая будет отличаться от исходной:

```
>>> str1 = 'Hello John'
>>> new_str = 'Welcome' + str1[5:]
>>> print(str1)
Hello John
>>> print(new_str)
Welcome John
>>>
```

## &#x20;Представление о списках

Python поддерживает структуру данных под названием list, которая представляет собой изменяемую упорядоченную последовательность элементов. Каждый элемент в этом списке называется item. Списки определяются путем вставки значений в квадратные скобки \[ ]. Каждому элементу списка присваивается номер, который мы называем позицией или индексом. Индексы начинаются с нуля, то есть первый индекс равен нулю, второй индекс равен 1 и т.д. Мы можем выполнять следующие операции со списками: индексацию, нарезку, добавление, умножение и проверку принадлежности.

Встроенная в Python функция length возвращает длину этого списка. В Python также есть функция поиска самого большого и самого маленького элемента списка. Списки могут быть нумерованными, строковыми или смешанными.

Ниже приведен код для создания списка:

```
l = list()
numbers = [10, 20, 30, 40]
animals = ['Dog', 'Tiger', 'Lion']
list1 = ['John', 5.5, 500, [110, 450]]
```

Здесь мы создали три списка: первый - это numbers, второй - animals, а третий - list1. Список внутри другого списка называется вложенным списком. Наш list1 является вложенным списком. Список, не содержащий элементов, называется пустым списком; вы можете создать его с пустыми скобками \[].

Как и следовало ожидать, вы можете присваивать переменным значения из списка:

```
>>> cities = ['Mumbai', 'Pune', 'Chennai']
>>> numbers_list = [75, 857]
>>> empty_list = []
>>> print (cities, numbers_list, empty_list)
['Mumbai', 'Pune', 'Chennai'] [75, 857] []
```

## &#x20;Доступ к значениям в списках

Мы можем получить доступ к значениям из списка, используя значения индекса. Мы укажем номер индекса в \[ i]. Индекс начинается с 0. Обратитесь к приведенному примеру:

```
#!/usr/bin/python3
cities = ['Mumbai', 'Bangalore', 'Chennai', 'Pune']
numbers = [1, 2, 3, 4, 5, 6, 7 ]
print (cities[0])
print (numbers[1:5])


Output:
Mumbai
[2, 3, 4, 5]
```

## &#x20;Обновление списков

Вы можете обновлять элементы списков, как показано в следующем коде:

```
#!/usr/bin/python3
cities = ['Mumbai', 'Bangalore', 'Chennai', 'Pune']
print ("Original Value: ", cities[3])
cities[3] = 'Delhi'
print ("New value: ", cities[3])


Output:
Original Value: Pune
New value: Delhi
```

## &#x20;Удаление элементов списка

Чтобы удалить элемент списка, вы можете использовать оператор del, если вы точно знаете, какие элементы вы удаляете. Вы можете использовать метод remove(), если вы точно не знаете, какие элементы следует удалить. Обратитесь к следующему примеру:

```
#!/usr/bin/python3
cities = ['Mumbai', 'Bangalore', 'Chennai', 'Pune']
print ("Before deleting: ", cities)
del cities[2]
print ("After deleting: ", cities)

Output:
Before deleting: ['Mumbai', 'Bangalore', 'Chennai', 'Pune']
After deleting: ['Mumbai', 'Bangalore', 'Pune']
```

## &#x20;Основные операции со списком

Существует пять основных операций со списком:

* Конкатинация
* Умножение
* Вычисление длины
* Членство
* Итерация

| **Description**  | **Expression**                              | **Result**                       |
| ---------------- | ------------------------------------------- | -------------------------------- |
| Конкатинация     |  \[30, 50, 60] + \['Hello', 75, 66]         |  \[30,50,60,'Hello',75,66]       |
| Членство         |  45 in \[45,58,99,65]                       |  True                            |
| Итерация         |  for x in \[45,58,99] : print (x,end = ' ') |  45 58 99                        |
|  Умножение       |  \['Python'] \* 3                           |  \['python', 'python', 'python'] |
| Вычисление длины |  len(\[45, 58, 99, 65])                     |  4                               |

## &#x20;Операции со списком

Оператор `+` объединяет списки:

```
>>> c = a + b
>>> print c
[30,50,60,'Hello',75,66]
```

Аналогично, оператор \* повторяет список заданное количество раз:

```
>>> [0] * 4
[0, 0, 0, 0]
>>> ['Python'] * 3
['python', 'python', 'python']
```

## &#x20;Кортежи

Структура данных кортежей в Python неизменяема, что означает, что мы не можем изменять элементы кортежей. По сути, кортеж - это последовательность значений, разделенных запятыми и заключенных в круглые скобки ( ). Как и списки, кортежи представляют собой упорядоченную последовательность элементов:

```
>>> t1 = 'h', 'e', 'l', 'l', 'o'
```

Кортежи заключаются в круглые скобки ( ):

```
>>> t1 = ('h', 'e', 'l', 'l', 'o')
```

Вы также можете создать кортеж из одного элемента. Вам просто нужно поставить последнюю запятую в кортеже:

```
>>> t1 = 'h',
>>> type(t1)
<type 'tuple'>
```

Значение, заключенное в круглые скобки, не является кортежем:

<pre><code><strong>>>> t1 = ('a')
</strong><strong>>>> type(t1)
</strong><strong>&#x3C;type 'str'>
</strong></code></pre>

Мы можем создать пустой кортеж, используя функцию tuple():

```
>>> t1 = tuple()
>>> print (t1)
()
```

Если аргументом является последовательность (строка, список или кортеж), результатом будет кортеж с элементами последовательности:

```
>>> t = tuple('mumbai')
>>> print t
('m', 'u', 'm', 'b', 'a', 'i')
```

Кортежи содержат значения, заключенные в круглые скобки ( ) и разделенные запятыми:

```
>>> t = ('a', 'b', 'c', 'd', 'e')
>>> print t[0]
'a'
```

Оператор slice выбирает диапазон элементов.

```
>>> print t[1:3]
('b', 'c')
```

## &#x20;Доступ к значениям в кортежах

Чтобы получить доступ к значениям в кортеже, используйте квадратные скобки для разбивки на части вместе с индексом или индексами, чтобы получить значение, доступное для этого индекса или индексов, как показано в следующем примере:

```
#!/usr/bin/python3
cities = ('Mumbai', 'Bangalore', 'Chennai', 'Pune')
numbers = (1, 2, 3, 4, 5, 6, 7)
print (cities[3])
print (numbers[1:6])

Output:
Pune
(2, 3, 4, 5)
```

## &#x20;Обновление кортежей

Обновление кортежей в Python невозможно, поскольку кортежи неизменяемы. Но вы можете создать новый кортеж из существующего кортежа, как показано в следующем примере:

```
#!/usr/bin/python3
cities = ('Mumbai', 'Bangalore', 'Chennai', 'Pune')
numbers = (1,2,3,4,5,6,7)
tuple1 = cities + numbers
print(tuple1)

Output:
('Mumbai', 'Bangalore', 'Chennai', 'Pune', 1, 2, 3, 4, 5, 6, 7)
```

## &#x20;Удаление элементов кортежа

Мы не можем удалить отдельные элементы кортежа. Поэтому, чтобы явно удалить весь кортеж, используйте инструкцию del. Обратитесь к следующему примеру:

```
#!/usr/bin/python3
cities = ('Mumbai', 'Bangalore', 'Chennai', 'Pune')
print ("Before deleting: ", cities)
del cities
print ("After deleting: ", cities)

Output:

Traceback (most recent call last):
File "01.py", line 5, in <module>
print ("After deleting: ", cities)

```

## max() и min()

Используя функции max() и min(), мы можем найти наибольшее и наименьшее значения из кортежа/списка. Эти функции позволяют нам получить информацию о количественных данных. Давайте рассмотрим пример:\


```
>>> numbers = (50, 80,98, 110.5, 75, 150.58)
>>> print(max(numbers))
150.58
>>>
```

Используя max(), мы получим наибольшее значение в нашем кортеже. Аналогичным образом мы можем использовать функцию min():

```
>>> numbers = (50, 80,98, 110.5, 75, 150.58)
>>> print(min(numbers))
50
>>>
```

Итак, здесь мы получаем минимальное значение.

## &#x20;Множества

Множество - это неупорядоченная коллекция элементов, не имеющих дубликатов. Основное назначение набора - проверка принадлежности и устранение повторяющихся записей. Эти объекты множества поддерживают математические операции, такие как объединение, пересечение, разность и симметричная разность. Мы можем создать набор, используя фигурные скобки или функцию set(). Если вы хотите создать пустой набор, то используйте set(), а не {}.

Вот краткая демонстрация:

```
>>> fruits = {'Mango', 'Apple', 'Mango', 'Watermelon', 'Apple', 'Orange'}
>>> print (fruits)
{'Orange', 'Mango', 'Apple', 'Watermelon'}
>>> 'Orange' in fruits
True
>>> 'Onion' in fruits
False
>>>
>>> a = set('abracadabra')
>>> b = set('alacazam')
>>> a
{'d', 'c', 'r', 'b', 'a'}
>>> a - b
{'r', 'd', 'b'}
>>> a | b
{'d', 'c', 'r', 'b', 'm', 'a', 'z', 'l'}
>>> a & b
{'a', 'c'}
>>> a ^ b
{'r', 'd', 'b', 'm', 'z', 'l'}
```

Понимание множества также поддерживается в Python. Обратитесь к следующему коду:

```
>>> a = {x for x in 'abracadabra' if x not in 'abc'}
>>> a
{'r', 'd'}
```

## &#x20;Словари

**Словарь** - это тип данных в Python, который состоит из пар ключ-значение и заключен в фигурные скобки {}. Словари неупорядочены и индексируются по ключам, где каждый ключ должен быть уникальным. Эти ключи должны быть неизменяемого типа. Кортежи можно использовать в качестве ключей, если они содержат только строки, числа или кортежи.

Просто пара фигурных скобок создает пустой словарь: {}. Основными операциями со словарем являются сохранение значения с некоторым ключом и извлечение значения, присвоенного ключу. Также можно удалить пару ключ-значение с помощью del. Если вы сохраняете данные с использованием ключа, который уже используется, старое значение, связанное с этим ключом, забывается. Извлечение значения с использованием несуществующего ключа приводит к ошибке. Вот небольшой пример использования словаря:

```
>>> student = {'Name':'John', 'Age':25}
>>> student['Address'] = 'Mumbai'
>>> student
student = {'Name':'John', 'Age':25, 'Address':'Mumbai'}
>>> student['Age']
25
>>> del student['Address']
>>> student
student = {'Name':'John', 'Age':25}
>>> list(student.keys())
['Name', 'Age']
>>> sorted(student.keys())
['Age', 'Name']
>>> 'Name' in student
True
>>> 'Age' not in student
False
```

Для создания словарей используются произвольные выражения ключей и значений, а также словарные описания:

```
>>> {x: x**2 for x in (4, 6, 8)}
{4: 16, 6: 36, 8: 64}
```

Когда ключи представляют собой простые строки, иногда проще указать пары, используя аргументы ключевых слов:

```
>>> dict(John=25, Nick=27, Jack=28)
{'Nick': 27, 'John': 25, 'Jack': 28}
```
