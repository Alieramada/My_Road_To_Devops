# 4.3 Где хранятся данные

## Ключевые области знаний

* Программы и конфигурация
* Процессы
* Адреса памяти
* Системный обмен сообщениями
* Ведение журнала

## Неполный список используемых файлов, терминов и утилит

* `ps`, `top`, `free`
* `syslog`, `dmesg`
* `/etc/`, `/var/log/`
* `/boot/`, `/proc/`, `/dev/`, `/sys/`
